# iMoon07 Portfolio & Blog Engine

Personal portfolio and interactive cybersecurity blog engine built for high-performance reading and content delivery.

## Overview

This repository hosts the source code for my personal portfolio and custom-built blog engine. It is designed to dynamically fetch, render, and display markdown-based technical write-ups and cybersecurity research from external repositories (such as my `Penjelajah-CyberSecurity` repository).

Access the live site here: https://imoon07.github.io/

## Core Features

- **Dynamic Markdown Rendering**: Translates raw `.md` files into beautifully styled HTML on-the-fly using Marked.js.
- **Bilingual Support**: Built-in routing and content switching mechanism for English and Indonesian articles.
- **Slug-based Routing**: Clean and professional URL structures (e.g., `?post=owasp-lab-toolkit&lang=en`).
- **Interactive Discussions**: Integrated with Giscus to provide a unified, GitHub-powered comment section across different language versions of the same article.
- **Theme Toggling**: Native dark mode and light mode support for comfortable reading.
- **Social Media Ready**: Fully optimized Open Graph meta tags for large banner unfurling on professional platforms like LinkedIn, Medium, Discord, and Telegram.

## Architecture

The project relies entirely on client-side vanilla web technologies (HTML, CSS, JavaScript) to ensure maximum speed, security, and simplicity without the need for complex backend frameworks or database setups.
