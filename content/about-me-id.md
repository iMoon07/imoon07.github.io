# Tentang Kurnia

Halo! Saya Raja Muhammad Kurnia Setyawan (@iMoon07). Di usia 21 tahun, saya berkarir sebagai **Cyber Security Architect** dan **Penetration Tester** dengan fokus utama pada *Offensive Security*.
