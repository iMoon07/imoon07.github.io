# Tentang Kurnia

Halo! Saya Raja Muhammad Kurnia Setyawan. Di usia 21 tahun, saya mulai berkarir sebagai **self-taught Penetration Tester** dengan fokus kuat pada *Offensive Security*.

## Awal Mula (Origin Story)

Ketertarikan saya pada dunia teknologi dimulai sejak bangku Sekolah Dasar (SD). Tumbuh sebagai "anak warnet", saya sangat gemar mengeksplorasi dunia *game* dan menggunakan *cheat*—sekadar sebagai pemain yang penasaran bagaimana batasan dan aturan sebuah sistem bisa diakali. Rasa penasaran ini menjadi alasan saya mengambil jurusan Teknik Komputer Jaringan (TKJ) di SMKN 1 Bintan Utara. 

Insting *ngoprek* saya semakin terasah ketika saya terjun secara otodidak memanajemen *server* bersama komunitas **Growtopia Private Server (GTPS)**. Sebagai konteks, Growtopia adalah *game 2D sandbox MMO* yang populer dengan sistem ekonomi dan *building*-nya. Sementara itu, GTPS adalah *private server* independen yang dibangun dari hasil *reverse engineering* atau *source code* yang bocor (*leak*). Terlibat di komunitas ini memberikan saya ruang bebas untuk bereksperimen dengan lalu lintas jaringan (*networking client-server*), modifikasi logika *game* (seringkali melibatkan C++), hingga administrasi server secara penuh.

Sebagai kenang-kenangan, ini adalah jejak digital (arsip YouTube) masa-masa saya dan kawan-kawan komunitas *ngoprek* server GTPS:

Kemudian di usia 20 tahun, saya akhirnya memiliki "mesin tempur" komputer pertama saya (12 vCPU, 16GB RAM, RTX 3060, 500GB SSD). Berbekal mesin ini, eksplorasi saya meledak. Saya aktif melahap berbagai program *bootcamp*—dimulai dari **Merdeka Siber**, lalu dilanjutkan dengan *bootcamp* kedua di **[Cyber Sentinel Secure](https://cybersentinelsecure.asia/)**. 

Bermodal ilmu dari pelatihan intensif tersebut, saya langsung mendapatkan kesempatan berharga untuk magang pertama kalinya di **PT. Teknologi Server Indonesia** ([xcode.co.id](https://xcode.co.id/)). Di sana, saya banyak berlatih fundamental *hacking* dan keamanan siber di bawah bimbingan para praktisi senior. Rangkaian pengalaman magang dan *bootcamp* ini terbukti menjadi katalisator yang kuat, hingga akhirnya saya mendarat di pekerjaan profesional pertama sebagai *Freelance Junior Pentester* di [Merdeka Siber](https://merdekasiber.com).

Perjalanan ini baru saja dimulai. Di usia 21 tahun ini, ruang untuk terus bereksplorasi masih terbentang luas, dan saya akan terus berkembang.
